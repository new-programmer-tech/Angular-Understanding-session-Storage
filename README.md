# Angular-Understanding-session-Storage

```
To add:-
 sessionStorage.setItem('shareHolder', JSON.stringify(data['name']));
```

```
TO Remove:-
sessionStorage.removeItem('name')
```

```
TO get:-
sessionStorage.getItem('name')

```
